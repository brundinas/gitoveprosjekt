# Miniprosjekt: Klassen vår

Dette prosjektet brukes til å øve på GitHub-samarbeid i par.

## Oppgave
Hver elev skal:
1. Lage en egen branch fra par-branchen
2. Legge til navnet sitt i `index.html`
3. Gi navnet sitt en egen CSS-klasse i `style.css`
4. Lage commit, push og pull request
5. Få partneren til å reviewe og merge inn i par-branchen
6. Lage en felles pull request fra par-branchen til `main`

## Eksempel
I `index.html`:
```html
<li class="elev-ola">Ola</li>
```

I `style.css`:
```css
.elev-ola {
  color: blue;
}
```

## Mål
- clone repo
- lage branch
- commit og push
- opprette pull request
- lese diff i "Files changed"
- reviewe og merge andres endringer
